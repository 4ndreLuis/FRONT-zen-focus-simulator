# ZenFocus Simulator

Um simulador de respiração funcional desenvolvido em TypeScript para praticar lógica assíncrona e manipulação de estilos dinâmicos.

## Desafios Superados

Este projeto foi um marco no meu aprendizado, onde foquei em:

- **Controle de Tempo:** Uso de `setInterval` e `clearInterval` para criar ciclos rítmicos.
- **Lógica de Alternância (Toggle):** Implementação de estados booleanos para alternar entre as fases de inspiração e expiração.
- **Feedback Visual Dinâmico:** Manipulação de propriedades CSS (`transform: scale` e `backgroundColor`) via TypeScript para criar uma experiência imersiva.

## Tecnologias

- TypeScript (Tipagem estrita e manipulação de DOM)
- CSS3 (Transições suaves e variáveis)
